⚠️ WARNING — Full Rescan uses a LOT of API calls!

  • This will delete ALL existing CW: tags from your entire library first
  • Then it will make 1 Groq API call per item — if you have 500 movies, that's 500 calls
  • Groq's free tier has rate limits — large libraries may hit the limit and fail mid-scan
  • You may need to re-run the scan multiple times on very large libraries
  • Consider using "Scan Untagged Items Only" above instead — it's much safer
